要件定義¶

機能要件¶

  • 認可リクエスト

    • 認可リクエストに必要となるパラメータをブラウザから入力できる

    • 認可レスポンスを受け取ることができる

    • 認可レスポンスの結果を画面に表示できる

  • トークンリクエスト

    • トークンリクエストを送ることができる

    • トークンレスポンスを受け取ることができる

    • トークンレスポンスの結果を表示することができる

  • リソースへの問合せ

    • 取得したアクセストークンを利用して、リソースを取得することができる

    • 取得したリソースを画面に表示することができる

  • 設定画面

    • デフォルトの認可リクエストの設定を保持することができる

    • 過去の認可レスポンス・トークンレスポンスを参照することができる

インフラ要件¶

  • 利用環境

    • AWS

  • Webサーバ

    • ApatchまたはNginx

  • アプリケーションサーバ

    • Tomcat

  • DB

    • MySQL
